# One Mermaidkai Theme

A fork of [One Monokai](https://github.com/azemoh/vscode-one-monokai) that flips around the yellow and green so that functions are yellow and strings are green.

## Install

press `ctl/command + shift + p` to launch the command palette then run
```
ext install one-monokai
```
